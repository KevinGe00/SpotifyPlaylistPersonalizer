# Spotify Personalizer

## Description
A web service allowing you to filter input playlists, providing you with only songs by artists that you listen to through a playlist instantly generated onto your Spotify library.

## Stack
```
HTML/CSS
BootStrap 
NodeJS
Express.js
OAuth 2.0
```
